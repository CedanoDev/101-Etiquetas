ETIQUETAS INICIALES O DE RAIZ

Etiqueta Descripción Ejemplo Elemento adicional
<!DOCTYPE html> Indica al navegador que el documento está basado en el estándar HTML5

<!DOCTYPE html>

Debajo de el se encuentran todas las etiquetas HTML
<html> </html> Representa la raíz de un documento HTML. Todos los demás elementos de la
estructura HTML deben ser recogidos dentro de estas etiquetas.

<html>
<head>
</head>
<body>
</body>
</html>

Dentro de estas etiquetas estaran las etiquetas body, head, y footer.

METADATOS DEL DOCUMENTO

Etiqueta Descripción Ejemplo Elemento adicional
<head> </head> Representa una colección de metadatos acerca del documento, incluyendo
enlaces a, o definiciones de, scripts y hojas de estilo.

<head>
<title>Ejemplo de código HTML</title>
<link rel="stylesheet" href="styles.css">
<script src="script.js"></script>
</head>

Dentro de esta etiqueta estaran las etiquetas title, link, script
entre otras.
<title> </title> Etiqueta usada para definir el título de la página web. <title>Ejemplo de código HTML</title>

Esta esta anidada dentro de la etiqueta head.
<link> Se usa para enlazar recursos externos al documento HTML. El ejemplo más
común son las hojas de estilos CSS.

<link rel="stylesheet" href="styles.css">

Es acompanada de los atributos rel, y href
<meta> Etiqueta usada para definir otros metadatos que no se pueden definir con
una etiqueta HTML especifica. Por ejemplo, para definir el autor del
sitio, o la descripción del mismo.

<meta charset="UTF-8">

Es acompanada de loa atributos charset, property y content.
<style> </style> Etiquetas usadas para introducir código CSS en línea, es decir, en el
propio documento HTML.

<style>
body {
background-color: lightblue;
}
h1 {
color: navy;
text-align: center;
}
</style>

Tiene atributos como background-color, color, tex-aling
entr otros

ETIQUETAS DE SECCIONES O PARA ESTRUCTURAR EL HTML

Etiqueta Descripción Ejemplo Elemento adicional
<body> </body> Al contrario que la etiqueta de metadatos , todo lo que quieras
mostrar en la página web debe ir recogido dentro de las etiquetas
de apertura y cierre de <body>. Este contenido será el que se muestre en la web.

<body>
<h1>¡Hola, mundo!</h1>
<p>Este es mi primer párrafo.</p>

<img src="mi_imagen.jpg" alt="Una imagen descriptiva">
</body>

Dentro de la etiqueta body van etiquetas como las de form, img, video, p, entre otras
<nav> </nav> Usadas para definir el contenido que será la sección de navegación de la web.

<nav>
<ul>
<li><a href="index.html">Inicio</a></li>
<li><a href="acerca-de.html">Acerca de</a></li>
<li><a href="contacto.html">Contacto</a></li>
</ul>
</nav>

Se pueden incluir listas ordenadas y des-ordenada (ul, ol)
<main> </main> Se usa para definir el contenido principal del documento. Solamente puede
existir uno por documento.

<main>
<h2>Contenido principal</h2>
<p>Este es el contenido principal de mi página web.</p>
<p>Aquí puedes agregar más texto, imágenes, videos, etc.</p>
</main>>

Elementos adicionales de esta etiqueta son etiquetas h... p,
entre otras
<section> <section> Define una sección del documento

<section>
<h2>Acerca de nosotros</h2>
<p>Somos una empresa dedicada a...</p>
<p>Nuestra misión es...</p>
</section>

Este es un elemento adicional
<address> </address> aporta información de contacto para su <article> más cercano o ancestro
<body> en el último caso lo aplica a todo el documento.

<address>
<a href="mailto:contacto@ejemplo.com">contacto@ejemplo.com</a>
Calle Falsa 123
Ciudad, País </address≷

Este es un elemento adicional
<article> </article> Define contenido independiente de la web.

<article> <h2>Mi primer artículo</h2> <p>Este es un artículo de ejemplo. Contiene información relevante
y puede ser compartido de forma independiente.</p> <p>Aquí puedes agregar más texto, imágenes, enlaces, etc.</p> </article>

Este es un elemento adicional
<aside> </aside> Dentro de estas etiquetas suele alojarse el contenido adicional de la web.
Suele ser contenido relacionado con la web pero de poca importancia

<article>
<h2>Mi artículo principal</h2>
<p>Este es el contenido principal de mi artículo.</p>
<aside>
<h3>Información adicional</h3>
<p>Aquí puedes agregar contenido relacionado, como una biografía
del autor, enlaces a artículos relacionados, anuncios o widgets.</p>
</aside>
</article>

Este es un elemento adicional
<h1> <h2> <h3> <h4> <h5> <h6> Son etiquetas HTML muy importantes, ya que son usadas para jerarquizar el
contenido de la web. Las etiquetas se usan para explicar
brevemente el contenido que irá a continuación.

<h1>Esto es un titulo nivel 1<h1>
<h2>Esto es un titulo nivel 2<h2>
<h3>Esto es un titulo nivel 3<h3>
<h4>Esto es un titulo nivel 4<h4>
<h5>Esto es un titulo nivel 5<h5>
<h6>Esto es un titulo nivel 6<h6>

Este es un elemento adicional
<header> </header> Se usan para definir la cabecera de la página web. Suele contener el
logotipo, menú de navegación, etc.

<header>
<h1>Mi Sitio Web</h1>
<nav>
<ul>
<li><a href="index.html">Inicio</a></li>
<li><a href="acerca-de.html">Acerca de</a></li>
<li><a href="contacto.html">Contacto</a></li>
</ul>
</nav>
</header>

Este es un elemento adicional
<footer> </footer> Usadas para definir el pie de página.

<footer>
<p>© 2023 Mi Sitio Web. Todos los derechos reservados.</p>
<nav>
<ul>
<li><a href="#">Contacto</a></li>
<li><a href="#">Política de privacidad</a></li>
</ul>
</nav>
</footer>

Este es un elemento adicional
